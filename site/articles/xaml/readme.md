# XAML Usage

RibbonControl is designed to feel natural in Avalonia XAML. The most important setup areas are:

- theme inclusion,
- namespace wiring,
- resource reuse,
- content slots,
- icon and overlay presentation.

Start with [Theme Usage and Resources](theme-usage-and-resources.md), then continue to [Icons and Content Customization](icon-and-content-customization.md).
